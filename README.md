# Pixelator

A minimalist macOS image editor for pixelating regions of images.

## About

Pixelator is a vibe coding project. Most of the code was generated by **OpenCode Zen**, with assistance from **Claude AI**. All code has been reviewed by the developer.

> **Note:** This project is still under active development.

## Purpose

Pixelator allows users to:
- Open an image
- Drag to select regions
- Apply pixelation to selected regions instantly
- Save the result

It's designed to be a simple, focused tool for one specific task — nothing more.

## Tech Stack

- **SwiftUI** — Pure SwiftUI, no AppKit
- **Core Image** — For pixelation effects
- **UniformTypeIdentifiers** — For file handling
- **No third-party dependencies**

## Building

Open `Pixelator.xcodeproj` in Xcode and run.
